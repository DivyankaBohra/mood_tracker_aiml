**AI/ML Mood Tracker Script**:

A simple console-based Python application that uses Machine Learning to analyze and track user moods from text input.

# DESCRIPTION:

The AI/ML Mood Tracker allows users to enter how they feel in natural language. The system uses a Naive Bayes model with NLP techniques to predict the user’s mood (Happy, Calm, Neutral, Anxious, Sad, Angry).

It stores mood entries with timestamps in a CSV file and provides summary statistics like most frequent mood and average mood score. The project demonstrates basic Artificial Intelligence and Python programming concepts.

# FEATURES:

Mood Detection:- Predicts mood using Machine Learning
Multiple Categories:- Happy, Calm, Neutral, Anxious, Sad, Angry
Data Storage:- Saves entries in "mood_log.csv"
Mood Summary:- Shows mood count and average score
Menu Interface:- Easy console-based navigation

# Prerequisites:

Python 3.x
scikit-learn library

# How to Use:

Save the file as "main.py"

Install library:
bash pip install scikit-learn

Run the program:
bash python main.py

# EXAMPLE:

Enter choice: 1

Describe how you feel: I feel happy today

Output:
Mood saved: Happy (95%)

# Important Note:

The program includes an exit option for safe termination.
All data is stored in a CSV file, so mood history is preserved.
